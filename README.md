###可配置项
  - 想要屏蔽的地区(如: 中国大陆)
  - 指定的电影类型(如: 
    "动作",
		"战争",
		"科幻",
		"悬疑",
		"犯罪",
		"恐怖",
		"惊悚",
		"冒险")
  - 每个类型的挖掘深度(如: 2)
  - 最喜爱的清晰度(如: 1080p)
  
###运行方法
  - 按照喜好配置config.json
  - 将btDigger.jar与config.json放置于同一目录下
  - 执行`java -jar btDigger.jar`
